# دليل استخدام Excel Template لـ Indirect PLO Assessment

## 📋 هيكل الملف

يجب أن يحتوي ملف Excel على **3 أعمدة**:

| العمود | الوصف | الصيغة | مثال |
|--------|--------|--------|--------|
| **A: PLO** | كود البرنامج التعليمي | `PLO-X.X` | PLO-1.1, PLO-2.3 |
| **B: Indirect Value (%)** | القيمة بين 0-100 | رقم أو نسبة مئوية | 75, 82.5, 68% |
| **C: Academic Year** | السنة الأكاديمية | `YYYY-YYYY` | 2024-2025 |

## ✅ صيغ صحيحة

```
PLO-1.1    75    2024-2025
PLO-2.3    82.5  2024-2025
PLO-3.1    68%   2024-2025
plo-1.2    70    2024-2025  (يتم تحويله إلى PLO-1.2 تلقائياً)
1.1        76    2024-2025  (يتم مطابقته مع PLO-1.1 تلقائياً)
```

## ❌ أخطاء شائعة

| الخطأ | الصحيح |
|--------|--------|
| `2024/2025` | `2024-2025` |
| `85.56%` | `85.56` (ستتم إزالة % تلقائياً) |
| خلايا فارغة | أملأ جميع الخلايا |
| `PLO 1.1` | `PLO-1.1` |

## 🚀 خطوات الاستخدام

1. **فتح الملف**: استخدم ملف `indirect-plo-template.xlsx` الموجود في مجلد `templates`
2. **تعديل البيانات**: غيّر البيانات الموجودة حسب احتياجاتك
3. **الحفظ**: احفظ الملف بصيغة Excel (.xlsx أو .xls)
4. **التحميل**: انتقل إلى تبويب PLOs وانقر على "Import Indirect PLO"
5. **اختر الملف**: اختر ملفك المعدّل وانقر "فتح"
6. **تأكيد**: سيتم عرض عدد السجلات المستوردة

## 📝 ملاحظات

- يتم تطبيع أكواد PLO تلقائياً (تحويل الأحرف الكبيرة، إزالة المسافات)
- يتم البحث عن PLO وإنشاء ما ينقص منها تلقائياً
- جميع القيم تُخزّن في قاعدة البيانات بشكل دائم
- النسب المئوية يتم تحويلها إلى أرقام عشرية (85% = 85)
